# Google Sheets Structure for Demo B

## Spreadsheet Setup

Create a new Google Sheet called "Agency Outreach Demo" with two tabs:
1. **Outreach Queue** — generated messages + classification
2. **Pipeline** — deal tracking by stage

---

## Tab 1: Outreach Queue

Use this tab structure for the n8n workflow to append and update rows.

### Headers (Row 1)

| Column | Header | Type | Notes |
|--------|--------|------|-------|
| A | Brand Name | Text | Primary key for updates |
| B | Handle | Text | Instagram handle without @ |
| C | Niche | Text | Swimwear, Resort Wear, etc. |
| D | DM | Text | Full Instagram DM generated by Claude |
| E | Email Subject | Text | Subject line from Claude |
| F | Email Body | Text | Email body from Claude |
| G | Status | Text | `Pending Review` → `INTERESTED` / `NOT_INTERESTED` / `FOLLOW_UP` |
| H | Generated At | DateTime | ISO timestamp when Claude generated |
| I | Reply | Text | Simulated or actual incoming reply |
| J | Classification | Text | `INTERESTED`, `NOT_INTERESTED`, `FOLLOW_UP`, or blank |

### Example Row (after first n8n run)

| A | B | C | D | E | F | G | H | I | J |
|---|---|---|---|---|---|---|---|---|---|
| Monday Swimwear | mondayswimwear | Swimwear | Hey Monday — your pieces have such a distinct identity... | Creator collab for Monday Swimwear | Hi Monday team, I run a boutique influencer agency... | Pending Review | 2026-05-12T14:23:45Z | Hey! We'd love to learn more about your creators... | INTERESTED |

## Tab 2: Pipeline

Optional tab for manual deal tracking. Structure suggested below.

### Headers

| Column | Header | Type | Notes |
|--------|--------|------|-------|
| A | Brand | Text | Brand name (reference to Outreach Queue) |
| B | Stage | Text | New Lead, Contacted, Replied, Qualified, Call Booked, Proposal Sent, Closed Won, Closed Lost |
| C | Owner | Text | Jason (or whoever is closing) |
| D | Next Action | Text | "Send follow-up", "Book call", "Send proposal", etc. |
| E | Last Updated | DateTime | When stage last changed |

### Example Rows

| A | B | C | D | E |
|---|---|---|---|---|
| Monday Swimwear | Replied | Jason | Book call | 2026-05-12 |
| Frankies Bikinis | New Lead | Jason | Send outreach | 2026-05-11 |

---

## Data Flow in n8n

1. **After Node 5 (Append to Sheets):** New row appears in Outreach Queue with Status = `Pending Review`
2. **After Node 9 (Update Sheets):** Existing row updates with:
   - Column I (Reply): simulated reply text
   - Column J (Classification): `INTERESTED`, `NOT_INTERESTED`, or `FOLLOW_UP`
   - Column G (Status): updates to match classification

---

## Optional: QUERY Formula for Pipeline Tab

To auto-populate the Pipeline tab with warm leads from Outreach Queue, use this formula in cell A2 of the Pipeline tab:

```
=QUERY('Outreach Queue'!A:J, "SELECT A,G,B WHERE J='INTERESTED'", 0)
```

This pulls all rows where Classification = INTERESTED into the Pipeline for manual follow-up.

---

## Row Numbering in n8n

When n8n appends a row, it goes below the headers (starting at row 2).
When n8n updates a row, it matches on column A (Brand Name).

Make sure n8n has edit access to the Sheet (Google Sheets OAuth2 credential).
